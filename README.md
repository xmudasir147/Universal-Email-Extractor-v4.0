# Universal-Email-Extractor-v4.0
Universal Email Extractor v4.0 — Developed a most powerful tool universal email extractor v4.0 to scrape emails, phone numbers, Facebook, Instagram, X (Twitter), LinkedIn, YouTube Channel, and contact page from any website.

# Key features:
- Get searching list of targeted websites from the user
- Find emails from provided websites, their sub-pages and social media pages
- Export the data on completion automatically, user can also export anytime manually
- Saves all time data into the SQLite3 Database
- Notify user on email about the progress of scraping if user is far away from the pc
- Notify user on email when it gets completed
- Auto updates system

# Technologies used:
- C#
- Selenium
- SQLite3
- DevExpress
- Windows Forms
- HTML Agility Pack
